# 設定ファイルについて

## プロジェクト設定 (project_info.csv)
- プロジェクト名: プロジェクトの識別名
- 解像度横: 出力動画の横幅(ピクセル)
- 解像度縦: 出力動画の縦幅(ピクセル)
- fps: フレームレート(1秒あたりのフレーム数)

## カット情報 (cut_info.csv)
- カット番号: 各カットの一意の識別子
- カットの尺(秒): カットの長さ(秒)
- カットの尺(フレーム): 秒未満のフレーム数
- ステータス: カットの状態(OK/NG等)
- テイク数: 撮影回数
- 担当者名: 担当者の名前

## 使用方法
1. 必要なサンプルファイル(*.sample.csv)を実際の設定ファイルにコピー
2. プロジェクトに合わせて内容を編集
3. ファイル名から".sample"を削除

## ディレクトリ構造
```
config/
├── project/              # プロジェクト設定
│   ├── project_info.csv     # 実際の設定
│   └── project_info.sample.csv  # サンプル
└── cut/                 # カット情報
    ├── cut_info.csv         # 実際の設定
    └── cut_info.sample.csv  # サンプル
